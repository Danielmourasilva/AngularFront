## Front-end Livraria

##  💻 Descrição

Este é um aplicativo de e-commerce de livros usando Angular.

###  📝 Executar API

```bash
   # Entre na pasta da API
   $ cd back-end
   
   #Instala dependências
   $ npm eu
   
   #Iniciar servidor
   $ npm início
```

###  📝 Executar aplicativo

```bash
   # Entre na pasta do aplicativo
   $ cd front-end
   
   #Instala dependências
   $ npm eu
   
   #Iniciar aplicativo
   $ npm início
```
